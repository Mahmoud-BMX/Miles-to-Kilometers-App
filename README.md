# Miles-to-Kilometers Converter

A simple Python app built with **ttkbootstrap** that converts miles to kilometers.  

## Features
- Modern and clean UI
- Supports **integer** and **float** inputs
- Instant conversion with a click

## How to Run

1. Clone the repo:

```bash
git clone https://github.com/<your-username>/miles-to-kilometers-app.git
```

2. Install dependencies:

```bash
pip install ttkbootstrap
```

3. Run the app:

```bash
python main.py
```

## Usage

* Enter a value in miles
* Click the arrow button
* The result in kilometers appears instantly
